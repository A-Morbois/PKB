# 201912161116 Git CheatSheet
tags = #Git , #Documentation, #IT


## Connaitre les branches présentes dans le répot
`git branch -a `

## Bouger de branche
`git checkout <Branch_Name>`

## Créer une branche 
`git checkout -b <Branch_Name>`


## See last commit
`$ git log --all --oneline --graph --max-count 20`
